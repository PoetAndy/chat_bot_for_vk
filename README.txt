Бот написан для социальной сети ВК.
Авторизация происходит по VK API.

Для генерации сообщения используется нейронная сеть WriteSonic,
запрос к котрой происходит по API, через request запрос.

Сайт неронной сети: https://writesonic.com/
Документация WriteSonic API: https://docs.writesonic.com/reference/introduction
Документация библиотеки vk_api: https://github.com/python273/vk_api
Официальная документация VK API по разделу сообщений: https://vk.com/dev/messages

Запуск бота осуществляется через основной файл main.py
Перед запуском необходимо:
    1) Загрузить необходимые библиотеки из файла requirements.txt
    2) Открыть файл .env и заполнить необходимые данные по примеру:
        ACCESS_TOKEN="API из ВК для авторизации, берется в настройках группы"
        WRITE_SONIC_URL="URL берестся в документации по интеграции неронной сети"
        X_API_KEY="API ключ получить можно в профиле на сайте неронной сети"
        USER_ID="ID пользователя по умолчанию для проверок"

Для настройки генерации ответа смотрим в конструкцию:
    payload = {
        "enable_google_results": "False",   | Для использования при генерации гугл запросов выставить (True / False)
        "enable_memory": True,              | Для использования памяти контекста диалога (True / False)
        "input_text": "Some question",      | Вопрос пользователя записывается сюда, изменения не требуются
        "history_data": [                   | Параметр history_data используется только при ("enable_memory": True)
            {
                "is_sent": True,            | True относится к пользователю
                "message": ""               | Сообщение пользователя
            },
            {
                "is_sent": False,           | False относится к сгенереррованному ответу
                "message": ""               | Сгенерированное сообщение
            }
        ]
    }
